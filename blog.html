3. What is event bubble?

The Event bubbling in JavaScript is a type of event propagation. The event triggers the innermost target element and consecutively triggers the parent element of the target element in the same hierarchy until it triggers the outermost element.

4. What is a callback function and why will you use it?

A callback function is one of the superpowers of JavaScript. It is the way JavaScript passes a function into another function as an argument. The callback function is called in the outer function to execute an action.
For example: showing a pop-up message welcoming a user after the user inputs a username.
